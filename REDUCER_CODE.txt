#!/usr/bin/env python
import sys

current = None
total = 0

for line in sys.stdin:
    product, value = line.strip().split('\t')
    value = int(value)

    if current == product:
        total += value
    else:
        if current:
            print(current + "\t" + str(total))
        current = product
        total = value

# Last product print
if current:
    print(current + "\t" + str(total))

🧠 Explanation (short & exam-friendly)

• The reducer receives sorted key-value pairs from the mapper.
• It groups data by product.
• It adds sales values for each product.
• It outputs total sales for each product.


📊 Example Output (Reducer)
A   450
B   600
C   450
D   250
E   400
