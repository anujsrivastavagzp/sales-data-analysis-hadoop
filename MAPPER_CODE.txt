#!/usr/bin/env python
import sys

for line in sys.stdin:
    # Skip header
    if "orderID" in line:
        continue

    data = line.strip().split(',')

    product = data[1]
    sales = int(data[4])

    print(product + "\t" + str(sales))


🧠 Explanation (short & exam-friendly)

• The mapper reads each line of input data.
• It skips the header row.
• It extracts the product name and sales value.
• It outputs data in key-value format (product, sales).


⚡ Example Output (Mapper)
A   100
B   200
A   150
