## 📌 Project Overview

In today’s data-driven world, businesses generate massive volumes of sales data daily. Traditional systems are not efficient enough to process such large-scale datasets.

This project presents a **scalable Big Data pipeline** using the Hadoop ecosystem to efficiently process and analyze sales data.

🔍 The system transforms raw data into **actionable insights**, such as:
- Product-wise total sales  
- Top-performing products  

---

## 🎯 Objectives

- Store large datasets using HDFS  
- Process data using MapReduce (Python)  
- Perform analytical queries using Hive  
- Understand distributed data processing  
- Generate insights from raw sales data  

## 🏗️ Architecture

<pre>
Raw CSV Dataset
        ↓
HDFS (Distributed Storage)
        ↓
MapReduce (Data Processing)
        ↓
Processed Data (HDFS)
        ↓
Hive (Data Analysis)
        ↓
Insights (Top Products)
</pre>

## ⚙️ Technologies & Tools

- **HDFS** – Distributed Storage  
- **MapReduce** – Parallel Processing  
- **Hive** – SQL-based Analysis  
- **Python** – Mapper & Reducer  
- **Linux / Cloudera** – Execution Environment  
- **ChatGPT** – AI Assistance

## 📊 Dataset Information

The dataset contains sales-related information:

| Attribute  | Description |
|------------|------------|
| orderID    | Unique order ID |
| product    | Product name |
| category   | Product category |
| region     | Sales region |
| sales      | Sales amount |
| quantity   | Quantity sold |
| discount   | Discount applied |
| profit     | Profit earned |

---

## 🔄 Workflow

1. Upload dataset to HDFS  
2. Process data using MapReduce  
3. Store processed output in HDFS  
4. Analyze data using Hive  
5. Generate insights
